# Скорость выполнения

```

	Код	= "
	|а=0;
	|Для н=1 По 10000 Цикл
	|	а=а+1;
	|КонецЦикла;";
	Результат	= "";
	Начало		= ТекущаяУниверсальнаяДатаВМиллисекундах(); //зафиксируем начало процесса...
	Выполнить(Код);
    Конец		= ТекущаяУниверсальнаяДатаВМиллисекундах(); //зафиксируем окончание процесса...
    Результат	= "" + (Конец - Начало) + " мс"; // /1000; 
    //вычислим время выполнения цикла в секундах.
```
