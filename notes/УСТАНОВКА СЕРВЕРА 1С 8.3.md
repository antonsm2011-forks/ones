1. Качаем отдельно  клиент 1с и сервер 1с (х64)
2. Устанавливаем клиент 1с
3. Устанавливаем сервер не забыв указать установку com и веб компонент
4. После установки произвести регистрацию
 4.1. regsvr32 "C:\Program Files\1cv8\8.3.xx.xxx\bin\comcntr.dll" (менять имя платформы в зависимости от версии)
 4.2. Регистрация утилиты администрирования серверов (8.3.xx.xxx) искать после установки или запустить RegMSC: C:\Program Files\1cv8\8.3.xx.xxx\bin
5. Проверить типовые обмены (работа COM компоненты)
6. Проверить Web сервер
 
ВАЖНО
При возникновении ошибок
Возможно потребуется удалить предыдущие версии (не проверено.)

При ошибке ММС смотреть пункт 4.2

Если не находит базы. А на сервере 1с пишет  ошибку кластера. Искать список баз в установленном сервер и 1с возможно они в другом месте.
(Может потребоваться перенести C:\Program Files (x86)\1cv8\srvinfo если вначале была установлен х86 сервер)

Включить http отладку:
Regedit
HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet (ControlSet001 ....) \Services\1C:Enterprise 8.3 Server Agent
"C:\Program Files (x86)\1cv8\8.3.11.2954\bin\ragent.exe" -debug -http -srvc -agent -regport 1541 -port 1540 -range 1560:1591 -d "C:\Program Files (x86)\1cv8\srvinfo"
